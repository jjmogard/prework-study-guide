HTML

  • !DOCTYPE html declaration tag at the beginning tells the browser what document type it should expect when opening
  • HTML is the basic frame and structure of the house
  • HTML attributes are always defined in the beginning element, never the closing element
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

  • Section elements are semantic that define a section in document that contains thematic content
  • id tags are used as a way to identify elements
  • ul stands for unordered list which uses bullet points to list items using the li tag inside it
  • li stands for list item which creates a new bullet point for each

CSS

  • CSS is the style and design of the webpage/document, much like the color, size, and shape of the house
  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding card designates this as a class selector. Any HTML element assigned to the class, card, will gain these property assignments from the CSS.

Git

  • git status: checks what branch we are currently on
  • git checkout main: switches to the main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • the git command must precede every Git statement
  • the checkout command is to move the working branch to a new branch
  • the -b flag creates a new branch
  • Git is software on your computer that tracks changes you make to your code
  • Github is a cloud-based service that allows developers to store, manage, and collaborate on code
  • SSH keys are unique identifiers for coders using git
  • Enter ssh-keygen -t ed25519 -C “your_email@example.com” in your terminal to create a new SSH key
  • Enter ls -al ~/.ssh in your terminal to see if any existing SSH keys are present. Look for a public SSH key that ends in .pub. For example, id_rsa.pub, id_ecdsa.pub, or id_ed25519.pub
